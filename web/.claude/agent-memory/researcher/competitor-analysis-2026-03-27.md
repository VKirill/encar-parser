## 1. CarsKorea.shop — Главная страница / Каталог

### Структура страницы (сверху вниз)

1. **Шапка (Header)**
   - Телефон 8 (800) — бесплатный по России
   - Ссылки на Telegram-канал, YouTube
   - Кнопка "Войти" (личный кабинет)
   - Навигация: Автомобили | Мотоциклы | Представительства | Аукцион | О нас | История продаж | Блог | Отзывы 5.0 | Схема привоза | FAQ | Лизинг | Статистика | Как проверить компанию? | Таможенный калькулятор
   - Блок "Курсы валют" (RUB/KRW, USD/KRW, CNY/KRW с датой обновления)
   - Мессенджеры: Чат в Telegram, WhatsApp

2. **H1:** "Автомобили из Южной Кореи под заказ с сайта Енкар (Encar)"
   - Ниже: промо-баннер к статье блога "Какие авто выгодно привезти из Кореи в 2026 году"

3. **Блок фильтров (полная форма)**
   - Марка (select2 dropdown с количеством авто в скобках, напр. "Hyundai (21228)")
   - Модель (зависимый от марки)
   - Конфигурация (зависимый от модели)
   - Год от/до (2018-2025)
   - Кузов (категории по размеру: Большие, Средние, Маленькие, RV, SUV...)
   - Коробка передач (АКПП, Вариатор, МКПП, Полуавтомат)
   - Двигатель (Дизель, Электричество, Бензин, Газ, гибриды)
   - Привод (2WD, 4WD)
   - Цвет (25+ вариантов)
   - Мощность (до 160 л.с.)
   - Экстерьер/интерьер (16 опций: люк, LED-фары, электробагажник и т.д.)
   - Безопасность (15 опций: подушки, ABS, TCS, ESC, камеры и т.д.)
   - Удобство/мультимедиа (17 опций: круиз, HUD, навигация, Bluetooth...)
   - Сидения (8 опций: кожа, электропривод, подогрев, вентиляция, массаж)
   - Кнопка "Раскрыть все параметры"
   - Кнопка "Показать X предложений" с динамическим счетчиком (93688)
   - Кнопка "Сбросить фильтр"

4. **Сетка марок** (быстрый доступ по брендам)
   - ТОП-15 популярных марок с количеством: Audi 2546 | BMW 8206 | Mercedes 8450 | Hyundai 21228 | Kia 24705 | и т.д.
   - "Все марки" — раскрывающийся список
   - Рекламный баннер "KoreaTop.ru — корейская косметика"

5. **Счетчик и сортировка**
   - "Показаны записи 1-20 из 93 688"
   - Сортировка: По актуальности | По возрастанию цены | По убыванию цены | По пробегу | По году выпуска: новее

6. **Карточки авто** (список, по 20 на странице)
   - Бейдж "Новое объявление"
   - Фото
   - Марка + Модель + Поколение, Год
   - **Цена: "Цена с ПТС во Владивостоке X ₽"** — КЛЮЧЕВАЯ ФИШКА
   - Конфигурация (полная), Тип топлива, Пробег
   - Таблица спецификаций: Конфигурация | Пробег | Двигатель | Выпуск
   - Кнопки: "Поделиться" (WhatsApp, Telegram, копировать ссылку) | "Открыть"

7. **Пагинация** (1-10 страниц)

8. **Форма обратной связи ("Остались вопросы?")**
   - Имя, Телефон, Город
   - "Отправить"

9. **Footer**
   - Навигация (дублирующая шапку)
   - Почта support@carskorea.shop
   - 5 офисов: Москва, СПб, Краснодар, Ростов-на-Дону, Хабаровск
   - Телефон 8 (800)
   - Соцсети
   - Юрлицо: ООО «Айти-результат», ИНН, КПП

### Сильные стороны CarsKorea (главная)
- Цена "с ПТС во Владивостоке" — УТП, снимает главное возражение "сколько в итоге"
- Количество авто в каждой марке — создает ощущение масштаба
- 93000+ объявлений — огромная база
- Множество фильтров, включая опции
- Курс валют прямо в шапке
- 5 офисов по России — доверие
- Юридические данные в футере

### Слабые стороны CarsKorea (главная)
- НЕТ hero-секции с value proposition
- НЕТ CTA в верхней части (только фильтры)
- НЕТ блоков доверия (процесс, гарантии, отзывы)
- НЕТ визуальных элементов (фото машин, lifestyle)
- Фильтры перегружены — слишком много параметров сразу
- Нет быстрого поиска (поисковая строка)
- Карточки текстовые, мало визуала
- Нет "Избранного" на главной (есть в меню, но не видно)
- Пагинация, а не infinite scroll

## 2. CarsKorea.shop — Страница модели (Volvo S90)

### URL: /volvo/s90/

### Структура
1. **Header** — идентичный главной
2. **H1:** "Купить Volvo S90 из Южной Кореи"
3. **Промо-баннер** к блогу (тот же)
4. **Фильтры** — те же что на главной, но предзаполнены:
   - Марка: Volvo
   - Модель: S90 (с подмоделями в dropdown)
   - Конфигурации: B5/B6/D4/D5/T5/T8 и т.д.
   - Счетчик: "Показать 206 предложений"
5. **Бейдж "Проходной"** — для авто 3-5 лет (самая низкая таможенная ставка)
   - С тултипом-объяснением
6. **Карточки** — те же что на главной, но только Volvo S90
7. **Бейдж "через X мес"** — для авто младше 3 лет (непроходной сейчас, но можно оставить на хранение)
   - Показывает 2 цены: текущую и будущую (когда станет проходным)
   - Тултип: "бесплатное хранение до 7 месяцев"

### Сильные стороны (страница модели)
- Бейдж "Проходной" — УНИКАЛЬНАЯ ФИШКА, очень полезная
- Возможность купить непроходной авто и подождать — интересная модель
- SEO title "Купить Volvo S90 из Южной Кореи с доставкой"
- SEO description с ключевиками

### Слабые стороны (страница модели)
- НЕТ описания модели (SEO-текст)
- НЕТ статистики по модели (средняя цена, диапазон, количество)
- НЕТ FAQ по модели
- НЕТ сравнения цен Корея vs Россия
- НЕТ отзывов покупателей этой модели
- По сути это просто отфильтрованный каталог, ничего уникального

### Сильные стороны Emavto.by
- Hero с четким value proposition ("Дилерские цены")
- Доказательство: "Поставляем для 30+ автосалонов" (социальное доказательство)
- "25% экономия" — конкретная цифра
- "Итоговая стоимость под ключ" — снимает страх скрытых расходов
- Разделение "Корейские / Импортные" — помогает ориентироваться
- Именные менеджеры с прямыми телефонами — персональный контакт
- Фильтры проще и понятнее чем у CarsKorea

### Слабые стороны Emavto.by
- Цены в USD, не в местной валюте — неудобно для покупателя
- НЕТ блоков доверия (процесс, гарантии, отзывы)
- НЕТ процесса работы на главной
- НЕТ отзывов клиентов
- Карточки очень минималистичные — мало информации
- Нет фото-галереи на карточках (одно фото)
- Нет мессенджер-кнопок (Telegram, WhatsApp)
- Дизайн устаревший
- Нет блога / контент-маркетинга
- Нет SEO-страниц по маркам/моделям
