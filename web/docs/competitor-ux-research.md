- **Расположение:** Вертикальный блок над каталогом (не sidebar, не top bar)
- **Принцип:** "Всё сразу" — все фильтры видны одним блоком, часть скрыта за "Раскрыть все параметры"
- **Марка/Модель/Конфигурация:** каскадные select2 dropdown-ы с количеством авто
- **Год:** от/до (2018-2025)
- **Остальные:** чекбоксы по группам
- **Динамический счетчик:** "Показать X предложений" обновляется при выборе
- **Сортировка:** отдельная строка над карточками (по актуальности, цене, пробегу, году)

- Характеристики строкой: "1.6 GDI Value Plus, Бензин, 126 895 км"
- Таблица: Конфигурация | Пробег | Двигатель | Выпуск

- 5 офисов по России (адреса)
- Юридические данные (ООО, ИНН, КПП)
- Отзывы 5.0 (ссылка в навигации)
- "Как проверить компанию?" (отдельная страница)
- Телефон 8(800) бесплатный
- Курсы валют актуальные (обновление каждый день)
- 93 688 объявлений (масштаб базы)

#### UX flow от входа до заявки:
1. Попал на главную -> видит фильтры -> выбирает марку/модель
2. Кнопка "Показать X предложений" -> обновляется список
3. Кликает "Открыть" на карточке -> переходит на детальную
4. На детальной: "Забронировать" или "Написать в Telegram/WhatsApp"
5. Альтернатива: форма "Остались вопросы?" внизу любой страницы

**Вывод:** Flow ориентирован на самостоятельного пользователя. Нет "воронки доверия" — человек сразу попадает в каталог без hero, без объяснения процесса, без гарантий. Работает только для тех, кто уже знает что такое CarsKorea.

#### Уникальные элементы:
- **Бейдж "Проходной"** — зеленый бейдж для авто 3-5 лет. Тултип: "Возраст автомобиля от 3-х до 5-ти лет, его можно привезти по самой низкой таможенной ставке". Крайне полезная информация.
- **Двойная цена для непроходных** — для авто <3 лет показывает текущую цену + будущую цену "через X мес" когда станет проходным. Тултип: "Можете приобрести сейчас и оставить на хранение (бесплатно до 7 месяцев)".

#### Чего нет (и должно быть):
- Описание модели Volvo S90 (SEO-текст)
- Средняя цена по модели, диапазон цен
- Количество доступных авто по годам
- Сравнение цен Корея vs Россия
- FAQ по модели
- Отзывы покупателей этой модели
- Похожие модели (S60, XC60 и т.д.)

**Вывод:** Страница модели = просто отфильтрованный каталог. Упущена SEO-возможность и конверсионный потенциал. Бейдж "Проходной" — единственный уникальный и полезный элемент.

## ЧАСТЬ 3. ЧТО ДОЛЖНО БЫТЬ НА КАЖДОЙ СТРАНИЦЕ

---

### 3.1. Главная страница (максимальная конверсия)

#### Hero-секция

**Рекомендуемая структура:**

```
[Бейдж] Напрямую из Кореи
[H1] Корейские авто на 30-50% дешевле российского рынка
[Sub] Hyundai, KIA, Genesis, BMW, Mercedes — с проверкой, доставкой и растаможкой под ключ.
      Вы платите только после полного осмотра автомобиля.
[CTA primary] Рассчитать стоимость
[CTA secondary] Смотреть каталог
[Trust stats] 2 800+ авто доставлено | Гарантия юр. чистоты | Доставка 14-21 день
[Справа/фон] Фото реального авто на фоне порта/контейнера
```

**Быстрый поиск в hero (альтернатива):**
Можно встроить мини-фильтр прямо в hero (как auto.ru):
```
[Марка ▼] [Модель ▼] [Бюджет от ___ до ___ ₽] [Найти]
```
Это сокращает путь от входа до каталога на один клик.

**Что НЕ делать:**
- Не начинать сразу с каталога без hero (как CarsKorea)
- Не делать hero без CTA (декоративный баннер)
- Не писать абстрактные слоганы ("Лучшие авто" — неинформативно)

#### Блоки после hero (рекомендуемый порядок):

1. **"Почему из Кореи"** — 3-4 карточки с иконками
   - "Экономия 30-50% от рыночной цены"
   - "Проверенное качество — 5 звезд EuroNCAP"
   - "Остаточная стоимость выше китайских на 15-20%"
   - "Прозрачная история: ДТП, страховые, пробег"

2. **Быстрый каталог / Популярные марки**
   - Сетка логотипов/плиток марок с количеством авто
   - ТОП-8 марок крупно: Hyundai, KIA, Genesis, BMW, Mercedes, Volvo, Porsche, Toyota
   - "Все марки" → каталог
   - У CarsKorea марки показаны текстом с числами — ЛУЧШЕ показать логотипами

3. **"Как это работает"** — таймлайн процесса (5 шагов)
   - Шаг 1: Подбор авто (3-7 дней)
   - Шаг 2: Осмотр и проверка в Корее
   - Шаг 3: Покупка и оформление
   - Шаг 4: Доставка морем (5-7 дней)
   - Шаг 5: Таможня + доставка до вас (7-20 дней)
   - Итого: 3-6 недель

4. **Каталог** (первые 8-12 карточек)
   - С фильтрами (базовые: марка, цена, год, сортировка)
   - Кнопка "Показать все X авто"

5. **"Сколько вы переплачиваете"** — сравнительная таблица
   - 4-5 популярных моделей: цена в России vs цена из Кореи под ключ
   - Разница выделена акцентным цветом
   - CTA: "Узнать цену на вашу модель"

6. **Калькулятор** (краткая версия)
   - Или ссылка "Рассчитать стоимость под ключ" с описанием что считает

7. **Отзывы** — карусель
   - Фото клиента + его авто
   - Текст отзыва
   - Модель, город доставки, срок
   - Рейтинг Яндекс.Карты (если есть)

8. **FAQ** — аккордеон с 5-7 вопросами
   - "Сколько стоит привезти авто из Кореи?"
   - "Сколько времени занимает доставка?"
   - "Какие гарантии?"
   - "Можно ли вернуть деньги?"
   - "Как проверяется автомобиль?"

9. **Форма заявки** (финальная секция)
   - "Получите бесплатный расчет стоимости"
   - Имя, Телефон, Комментарий (опционально)
   - "Перезвоним в течение 30 минут"

#### Показ брендов:

**Лучший подход:** Сетка логотипов/плиток, а не текстовый список.

```
[Hyundai logo]  [KIA logo]    [Genesis logo]  [BMW logo]
  21 228          24 705          7 932          8 206

[Mercedes]     [Volvo]        [Porsche]       [Toyota]
  8 450          1 035          1 440            409

[Все 50+ марок →]
```

Почему:

- Опции (люк, камера и т.д.) — слишком детально для каталога

- Область в Корее — не релевантно покупателю

#### Расположение фильтров:

**Рекомендация: Горизонтальная полоса сверху (sticky на desktop)**

Почему НЕ sidebar:
- На мобильных sidebar превращается в бесполезную кнопку "Фильтры"
- При 8-10 фильтрах sidebar занимает слишком много пространства
- Для этого проекта (основной фильтр = марка + цена) горизонтальный формат оптимален
- CarsKorea использует вертикальный блок — работает, но занимает много места

Почему НЕ только top:
- Для расширенных фильтров (10+) нужна панель или bottom sheet
- Sticky top bar с 5-6 фильтрами + кнопка "Ещё" — оптимальный баланс

#### Счетчики результатов:
- "Найдено 206 автомобилей" — всегда видно
- Обновляется динамически при изменении фильтров
- При пустом результате: "Ничего не найдено. Попробуйте изменить параметры или оставьте заявку на подбор"

#### Infinite scroll vs пагинация:

**Рекомендация: Пагинация** (как у обоих конкурентов)

Почему:
- SEO: каждая страница индексируется
- UX: пользователь может вернуться к "тому самому авто на 3 странице"
- Производительность: не грузить 93000 карточек бесконечно
- Совместимость: работает на слабых устройствах
- Стандарт отрасли: auto.ru, drom.ru, все дилеры — пагинация

По 20-24 карточки на страницу (как CarsKorea). 50 на страницу (как Emavto) — слишком много.

- После каждых 8-12 карточек — inline CTA-баннер
  - "Не нашли нужный? Подберём за 24 часа" → форма
  - Или "Рассчитайте стоимость под ключ" → калькулятор

| **Разбивка цены** (авто + таможня + доставка) | Нет | Нет | **ДА — единственные на рынке** |
| **Hero с value proposition** | Нет | Да, но базовый | Лучше: с цифрами и CTA |
| **Современный дизайн** | Bootstrap, устаревший | Устаревший | Tailwind, тёмная тема, DM Sans |
| **Быстрый поиск в hero** | Нет | Нет | Мини-фильтр прямо в hero |
| **Floating мессенджер** | Нет на всех страницах | Нет | Да, на каждой странице |
| **Sticky цена на детальной** | Нет (уезжает при скролле) | Нет данных | Да, sticky panel |
| **Фотогалерея с fullscreen** | Базовая | Минимальная | Полноценная с тумбнейлами |
| **Калькулятор на детальной** | Отдельная страница только | Нет | Встроен прямо на странице |
| **"Почему этот авто"** | Нет | Нет | Автогенерируемые highlight-ы |
| **Процесс на главной** | Нет (ссылка) | Нет | Визуальный таймлайн |
| **Mobile-first UX** | Desktop-first | Desktop-first | Mobile-first |
| **Скорость загрузки** | SSR, но медленный | CSR | Next.js SSR/ISR, оптимизация |
